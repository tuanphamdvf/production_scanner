Production_barcode

Problem statement: There are production systems where the personnel in charge of the process cannot use a computer, but you still want to manage their work processes and interact with the production system (e.g., start production, complete production, stop production, resume production).
Solution: We have developed connections from the Odoo system to the outside, where you can interact with production commands. As a result, we have built a simple mobile app where your employees can scan a start and completion code to perform their tasks.

Process details: (create product serial => scan serial (create production order) => scan code (start process 1) => scan code (complete process 1) => scan (complete production order)). For more details, please refer to the video demo in the description...
